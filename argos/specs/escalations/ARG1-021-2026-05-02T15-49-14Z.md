**Decision:** Option B ratified as the eventual direction, with the mechanism
generalized from the gitattributes-inspection framing in this escalation to
dynamic dry-run merge. ARG1-021 stays strict on main per the escalation's own

cleanly only via human-chosen registration pattern). The ratified mechanism
exercises actual merge behavior via dry-run, subsuming both the custom-driver
case (STATE.md) and the registration-pattern case (`__main__.py`).
